# Phase Roadmap

## Phase 1 — Discovery

Goal: Get the user from brand input to confirmed scan ingredients.

Outputs:
- Confirmed BrandProfile
- Confirmed Market
- Products/Services
- Topics
- Competitors if selected
- Audiences if available
- TrustSignals
- Discovery complete state

## Phase 2 — Tracker Setup & Execution

Goal: Convert Discovery outputs into a Visibility Tracker and execute scans.

Outputs:
- TrackerConfiguration
- Prompt Library / Prompt Templates usage
- Draft and Active Prompts
- Platform/cadence setup
- ScanRuns
- PromptRuns
- AIAnswers
- Raw provider response storage
- Live progress events
- Scheduled email recipient settings

## Phase 3 — Analysis & Findings

Goal: Convert raw AI answers into structured signals, metrics, findings, and content actions.

Outputs:
- AnswerSignals
- Mentions
- Sources/Citations
- Source classification
- ScanMetrics
- Findings
- ContentActions

## Phase 4 — Reporting & Actions UI

Goal: Present findings, metrics, actions, and evidence in a user-friendly, action-first reporting UI.

Outputs:
- Scan Results page
- Visibility Tracker dashboard
- Topic view
- Competitor view
- Source/Citation view
- Prompt evidence view
- Manual PDF report
- Scheduled email summary

## Later phases / deferred

- Focus Areas as ongoing improvement initiatives
- Content Briefs
- Content generation
- Workflow engine
- CMS/social publishing
- Approval workflow
- Integrations: GSC, GA4, CMS, SEO tools
- Pricing/plans
- Advanced event schema
- Advanced benchmarking
